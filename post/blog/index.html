用hugo搭建自己的博客

Dec 4, 2019
1 min read

1.安装git，hugo

2.用hugo来生成博客：hugo new site myblog

3.下载并设置主题：

  • https://themes.gohugo.io/

4.在本地启动个人博客：‘hugo server -t 主题名 –buildDrafts’

5.世纪写一篇文章：hugo new post/blog.md

  • (可以用visual studio code打开md文件修改文章内容)
  • hugo server -t 主题名 - -buildDrafts(保存) 注意要切换到myblog目录

6.将个人博客部署到远端服务器（github）：

  • 登录github，建一个new respository
  • !!! repository name 必须是自己的用户名（全小写）
  • eg：wsjjllk.github.io
  • 部署：hugo - -theme=主题名 - -baseUrl=”……” - -buildDrafts
  • 会生成public文件夹
  • 接着： git init
  • git add
  • git commit -m” 说明 "
  • 然后关联github：git remote add origin https:// .git
